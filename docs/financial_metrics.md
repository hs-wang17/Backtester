## 1 收益类指标

### 1.1 年化收益 (Annualized Return)

**定义**：策略在一年时间内的预期收益率。

**计算公式**：
$$\text{年化收益} = \text{日收益率均值} \times 250$$

其中，$250$ 为假设的年交易日数。

**理论背景**：
年化收益是衡量投资策略表现的基本指标，它将不同时间跨度的收益率标准化为年度收益率，便于跨策略比较。

**实际应用**：

- 用于评估策略的盈利能力
- 作为其他风险调整收益指标的基础
- 帮助投资者设定合理的收益预期

### 1.2 累计收益 (Cumulative Return)

**定义**：整个回测期间的总收益率。

**计算公式**：
$$\text{累计收益} = \frac{\text{期末净值}}{\text{期初净值}} - 1$$

**理论背景**：
累计收益反映了投资者在整个投资期间获得的实际收益，是最直观的盈利指标。

**实际应用**：

- 评估策略的长期表现
- 与基准进行比较
- 计算投资者的实际财富增长

### 1.3 超额年化收益 (Excess Annualized Return)

**定义**：策略相对于基准的超额收益的年化值。

**计算公式**：
$$\text{超额收益} = \text{策略日收益率} - \text{基准日收益率}$$
$$\text{超额年化收益} = \text{超额收益均值} \times 250$$

**理论背景**：
超额收益衡量了策略相对于市场基准的增值能力，是主动管理策略的核心评价指标。

## 2 风险类指标

### 2.1 年化波动 (Annualized Volatility)

**定义**：策略收益率的标准差，年化后的值。

**计算公式**：
$$\text{年化波动} = \text{日收益率标准差} \times \sqrt{250}$$

**理论背景**：
波动率是衡量投资风险的主要指标，反映了收益率的不确定性。年化波动使用平方根时间法则进行转换。

**实际应用**：

- 评估策略风险水平
- 计算风险价值(VaR)
- 构建投资组合优化模型

### 2.2 下行波动率 (Downside Volatility)

**定义**：仅考虑负收益率的波动率。

**计算公式**：
$$\text{下行波动} = \text{负收益率标准差} \times \sqrt{250}$$

**理论背景**：
下行波动率关注投资者的损失风险，更符合投资者对风险的真实感受。

**实际应用**：

- 计算 Sortino 比率
- 评估策略的下行风险
- 构建下行风险优化模型

### 2.3 VaR(95%) (Value at Risk)

**定义**：在 95%置信水平下，投资组合可能面临的最大损失。

**计算公式**：
$$\text{VaR(95\%)} = \text{收益率的5\%分位数}$$

**理论背景**：
VaR 是衡量市场风险的重要指标，回答了"在正常市场条件下，投资组合在给定时间内可能面临的最大损失"这一问题。

**实际应用**：

- 风险限额管理
- 监管资本计算
- 投资组合风险控制

### 2.4 CVaR(95%) (Conditional Value at Risk)

**定义**：在超过 VaR 的条件下，损失的期望值，也称为期望短缺(Expected Shortfall)。

**计算公式**：
$$\text{CVaR(95\%)} = \mathbb{E}[R | R \leq \text{VaR(95\%)}]$$

**理论背景**：
CVaR 解决了 VaR 不满足次可加性的问题，提供了更一致的风险度量。

**实际应用**：

- 更准确的风险评估
- 风险预算分配
- 压力测试

## 3 风险调整收益指标

### 3.1 夏普比率 (Sharpe Ratio)

**定义**：单位总风险获得的超额收益。

**计算公式**：
$$\text{夏普比率} = \frac{\text{年化收益}}{\text{年化波动}}$$

**理论背景**：
由诺贝尔奖得主 William Sharpe 提出，是衡量风险调整收益的基准指标。

**实际应用**：

- 比较不同策略的风险调整收益
- 评估投资经理的选股能力
- 构建有效前沿

### 3.2 Sortino 比率 (Sortino Ratio)

**定义**：单位下行风险获得的超额收益。

**计算公式**：
$$\text{Sortino比率} = \frac{\text{年化收益}}{\text{下行波动}}$$

**理论背景**：
Sortino 比率只考虑下行风险，更符合投资者对风险的真实感受。

**实际应用**：

- 评估策略的下行风险调整收益
- 适用于风险厌恶型投资者
- 补充夏普比率的不足

### 3.3 Calmar 比率 (Calmar Ratio)

**定义**：年化收益与最大回撤的比率。

**计算公式**：
$$\text{Calmar比率} = \frac{\text{年化收益}}{\text{最大回撤}}$$

**理论背景**：
Calmar 比率关注收益与最大损失的关系，适合评估回撤较大的策略。

**实际应用**：

- 评估策略的回撤恢复能力
- 适用于趋势跟踪策略
- 长期投资者风险评估

### 3.4 信息比率 (Information Ratio)

**定义**：超额收益与跟踪误差的比率。

**计算公式**：
$$\text{信息比率} = \frac{\text{超额年化收益}}{\text{跟踪误差}}$$

**理论背景**：
信息比率衡量主动管理策略相对于基准的价值增值能力。

**实际应用**：

- 评估主动管理效果
- 基金业绩评价
- 资产配置决策

### 3.5 Treynor 比率 (Treynor Ratio)

**定义**：超额收益与 Beta 的比率。

**计算公式**：
$$\text{Treynor比率} = \frac{\text{年化收益}}{\text{Beta}}$$

**理论背景**：
由 Jack Treynor 提出，衡量单位系统性风险获得的超额收益。

**实际应用**：

- 评估系统性风险管理能力
- 适用于充分分散的投资组合
- CAPM 框架下的业绩评估

### 3.6 Omega 比率 (Omega Ratio)

**定义**：收益分布中收益超过阈值的概率加权与低于阈值的概率加权之比。

**计算公式**：
$$\text{Omega比率} = {\int_{\theta}^{\infty} (1 - F(r)) \mathrm{d}r}\bigg/{\int_{-\infty}^{\theta} F(r) \mathrm{d}r}$$

其中 $\theta$ 通常设为 $0$，$F(r)$为收益率的累积分布函数。

**理论背景**：
Omega 比率考虑了收益分布的全部信息，不依赖于正态分布假设。

**实际应用**：

- 非正态分布收益评估
- 高阶矩风险考虑
- 衍生品策略评价

## 4 回撤相关指标

### 4.1 最大回撤 (Maximum Drawdown)

**定义**：策略净值从历史高点到低点的最大跌幅。

**计算公式**：
$$\text{回撤}_t = \text{历史最高净值}_t - \text{当前净值}_t$$
$$\text{最大回撤} = \max(\text{回撤序列})$$

**理论背景**：
最大回撤反映了投资者可能面临的最大损失，是风险控制的重要指标。

**实际应用**：

- 风险管理
- 投资者心理承受能力评估
- 策略参数调优

### 4.2 平均回撤 (Average Drawdown)

**定义**：所有回撤期间的平均跌幅。

**计算公式**：
$$\text{平均回撤} = \frac{1}{n} \sum_{i=1}^{n} \text{回撤}_i$$

其中只考虑回撤序列中的正值部分。

**理论背景**：
平均回撤提供了更稳定的回撤度量，减少了极端值的影响。

**实际应用**：

- 评估策略的平均风险水平
- 比较不同策略的回撤特征
- 风险预算制定

### 4.3 最长回撤持续天数 (Maximum Drawdown Duration)

**定义**：策略净值从高点恢复到新高所需的最长时间。

**计算公式**：
$$\text{最长回撤持续天数} = \max(\text{连续回撤天数序列})$$

**理论背景**：
回撤持续时间反映了资金的占用时间和投资者的等待成本。

**实际应用**：

- 流动性风险管理
- 投资者持有期规划
- 策略稳定性评估

### 4.4 超额最大回撤 (Excess Maximum Drawdown)

**定义**：超额收益曲线的最大回撤。

**计算公式**：
$$\text{超额净值}_t = 1 + \sum_{i=1}^{t} \text{超额收益}_i$$
$$\text{超额最大回撤} = \max(\text{超额净值的历史最高点} - \text{超额净值})$$

**理论背景**：
超额最大回撤衡量了相对基准的最大损失，评估主动管理风险。

**实际应用**：

- 评估相对风险
- 基准比较分析
- 主动风险控制

## 5 统计分布指标

### 5.1 偏度 (Skewness)

**定义**：收益率分布的不对称性度量。

**计算公式**：
$$\text{偏度} = \frac{\mathbb{E}[(R - \mu)^3]}{\sigma^3}$$

**理论背景**：
偏度反映了收益分布的尾部特征：

- 正偏度：右尾较长，可能出现极端正收益
- 负偏度：左尾较长，可能出现极端负收益

**实际应用**：

- 风险管理
- 期权定价
- 投资组合构建

### 5.2 峰度 (Kurtosis)

**定义**：收益率分布的尾部厚度度量。

**计算公式**：
$$\text{峰度} = \frac{\mathbb{E}[(R - \mu)^4]}{\sigma^4}$$

**理论背景**：
峰度反映了极端事件发生的概率：

- 正态分布：峰度=3
- 超额峰度>0：厚尾，极端事件概率高
- 超额峰度<0：薄尾，极端事件概率低

**实际应用**：

- 极端风险预警
- 风险模型校正
- 衍生品定价

## 6 市场相关指标

### 6.1 Beta 系数

**定义**：策略收益率相对于市场收益率的敏感性。

**计算公式**：
$$\text{Beta} = \frac{\text{Cov}(R_p, R_m)}{\text{Var}(R_m)}$$

**理论背景**：
Beta 是 CAPM 模型的核心参数，衡量系统性风险暴露。

**实际应用**：

- 系统性风险管理
- 资产配置
- 业绩归因分析

### 6.2 Alpha 系数

**定义**：策略收益中无法由市场风险解释的部分。

**计算公式**：
$$\text{Alpha} = \text{年化收益} - \text{Beta} \times \text{市场年化收益}$$

**理论背景**：
Alpha 代表了投资经理的选股能力或信息优势。

**实际应用**：

- 评估投资经理能力
- 策略有效性检验
- 业绩归因分析

### 6.3 跟踪误差 (Tracking Error)

**定义**：超额收益的标准差，年化后的值。

**计算公式**：
$$\text{跟踪误差} = \text{超额收益标准差} \times \sqrt{250}$$

**理论背景**：
跟踪误差衡量了策略相对于基准的偏离程度。

**实际应用**：

- 指数基金管理
- 风险预算控制
- 基准选择

## 7 时间序列分析指标

### 7.1 Hurst 指数 (Hurst Exponent)

**定义**：衡量时间序列长期记忆性的指标。

**计算公式**：
$$H = \frac{\log(R/S)}{\log(n)}$$

其中 $R/S$ 统计量通过重标极差法计算，$n$ 为时间间隔。

**理论背景**：
Hurst 指数反映了时间序列的记忆性：

- $H > 0.5$：持续性趋势
- $H = 0.5$：随机游走
- $H < 0.5$：均值回归

**实际应用**：

- 趋势识别
- 市场有效性检验
- 策略参数优化

### 7.2 收益稳定性 (Return Stability)

**定义**：对数净值线性拟合的决定系数。

**计算公式**：
$$R^2 = 1 - \frac{\sum_{i=1}^{n}(y_i - \hat{y}_i)^2}{\sum_{i=1}^{n}(y_i - \bar{y})^2}$$

其中$y_i$为实际对数净值，$\hat{y}_i$为拟合值，$\bar{y}$为均值。

**理论背景**：
$R^2$ 衡量了收益增长的稳定性和可预测性。

**实际应用**：

- 策略稳定性评估
- 增长趋势分析
- 风险管理

## 8 实用性指标

### 8.1 胜率(天) (Win Rate)

**定义**：正收益率天数占总交易天数的比例。

**计算公式**：
$$\text{胜率} = \frac{\text{正收益率天数}}{\text{总交易天数}}$$

**理论背景**：
胜率反映了策略的正确性概率。

**实际应用**：

- 策略信心建立
- 仓位管理
- 心理承受能力评估

### 8.2 最大连续亏损天数 (Maximum Consecutive Loss Days)

**定义**：策略连续出现亏损的最大天数。

**计算公式**：
$$\text{最大连续亏损天数} = \max(\text{连续负收益天数序列})$$

**理论背景**：
连续亏损反映了策略的阶段性失效风险。

**实际应用**：

- 资金管理
- 策略调整时机
- 心理风险管理
